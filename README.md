Expense Tracker — трекер расходов
Автор: Магамдалиев Марат Джалалович

Описание программы
Графическое приложение для учёта личных расходов. Позволяет добавлять траты с указанием суммы, категории и даты. Данные сохраняются в файл JSON. Доступна фильтрация по категории и дате, а также подсчёт общей суммы расходов за выбранный период.

Требования к окружению
Python 3.8+
Библиотека tkinter (входит в стандартную поставку Python).

Инструкция по запуску
Установите Python (если не установлен).
Скачайте или клонируйте репозиторий.
Запустите файл main.py командой в терминале:
python main.py
Приложение откроется в новом окне.

Использование программы
1. Введите сумму (положительное число), категорию и дату в формате ГГГГ-ММ-ДД.
2. Нажмите «Добавить расход» — запись появится в таблице.
3. Для фильтрации выберите категорию и/или укажите период (даты «с» и «по»), затем нажмите «Фильтровать».
4. Для подсчёта суммы за период заполните поля дат и нажмите «Сумма за период».
5. Все данные сохраняются автоматически при добавлении новой записи.

Примеры использования
Добавить расход: сумма — 1200.50; категория — «Еда»; дата — 2026-04-30.
Фильтр по категории: выберите «Транспорт» — в таблице останутся только траты на транспорт.
Подсчёт суммы: введите даты с 2026-04-01 по 2026-04-30 и нажмите «Сумма за период».

Тестирование (примеры)
Позитивный: ввод корректных данных (например: 500; Еда; 2026-04-29).
Негативный: ввод отрицательной суммы или даты в неверном формате — появится сообщение об ошибке.
Граничный: ввод очень большой суммы или пустой категории — приложение не позволит сохранить запись.
