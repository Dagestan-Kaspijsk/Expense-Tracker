Описание программы
«Expense Tracker» — это графическое приложение для учёта личных расходов. Позволяет добавлять расходы с указанием суммы, категории и даты, фильтровать записи по категории и периоду, а также подсчитывать общую сумму расходов за выбранный интервал. Данные сохраняются в файл data.json.

Требования для запуска
Python 3.10 или выше.
Библиотеки:
pip install PyQt6


Как запустить
1. Клонируйте репозиторий:
git clone https://github.com/твой-логин/expense-tracker.git
cd expense-tracker

2. Установите зависимости:
pip install -r requirements.txt

3. Запустите приложение:
python main.py

Как пользоваться программой
Введите сумму расхода в поле «Сумма».
Выберите категорию из списка.
Укажите дату расхода.
Нажмите «Добавить расход» — запись появится в таблице.
Для фильтрации выберите категорию и/или укажите период (даты «С» и «По»), затем нажмите «Фильтровать».
Для подсчёта суммы за период укажите даты и нажмите «Сумма за период».
Все данные автоматически сохраняются в data.json.
Пример работы программы
Добавление расхода:

Сумма: 1500
Категория: Еда
Дата: 2026-04-29
Нажать «Добавить расход» — запись добавлена.Фильтрация:
Категория: Транспорт
Период: с 2026-04-01 по 2026-04-30
Нажать «Фильтровать» — отображаются только траты на транспорт за апрель.Подсчёт суммы:
Период: с 2026-04-01 по 2026-04-30
Нажать «Сумма за период» — появится сообщение с итоговой суммой.
